# Enterprise CRM System 🚀

A full-stack Enterprise CRM System built using React.js, Node.js, Express.js, and MongoDB for managing leads, customers, and business workflows.

## 🔥 Features

- User Authentication System
- CRM Dashboard
- Lead Management
- REST API Integration
- Backend Routing & Controllers
- Secure Environment Variables
- Full Stack Architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## ▶️ Run Frontend

```bash
cd client
npm start
```

Frontend URL:

```bash
http://localhost:3000
```

---

## ▶️ Run Backend

```bash
cd server
npm run dev
```

Backend URL:

```bash
http://localhost:5000
```

---

## 🔐 Environment Variables

Create `.env` inside server folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
